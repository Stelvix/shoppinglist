# Shopping List API

## Présentation
Shopping List API est une API qui permet de gérer une liste de courses simplement.
Elle sert à ajouter, consulter, modifier et supprimer des produits, afin de garder une liste à jour pour ses achats du quotidien.

## Ce que fait l’API
L’API permet de :
- consulter tous les produits de la liste ;
- ajouter un nouveau produit ;
- consulter le détail d’un produit ;
- modifier les informations d’un produit ;
- supprimer un produit de la liste ;
- consulter les types de courses (catégories) disponibles.

## Pourquoi c’est utile
Cette API aide à :
- organiser ses courses de manière claire ;
- suivre facilement les produits à acheter ;
- garder une liste propre en retirant les produits non nécessaires ;
- structurer les achats par catégories.

## Exemple d’usage
Un utilisateur prépare ses courses de la semaine :
1. il ajoute les produits dont il a besoin ;
2. il met à jour certains prix ;
3. il supprime les produits déjà achetés ;
4. il consulte les catégories pour mieux organiser son passage en magasin.

## Vérification rapide
Un point d’entrée simple permet de vérifier que l’API répond correctement (`/hello`).

## Objectif
L’objectif du projet est de proposer une base simple et pratique pour une application de gestion de courses (web ou mobile).
